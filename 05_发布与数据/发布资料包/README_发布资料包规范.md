更新日期：2026-07-09

目标：每条视频发布前，都有一个可直接复制使用的发布资料包，避免标题、简介、话题、来源备注散在聊天里。

## 文件命名

```text
YYYY-MM-DD_第N期_发布资料包.md
```

可直接复制模板：

```text
TEMPLATE_发布资料包.md
```

## 平台差异

- 视频号：标题和简介要更直白，避免过度标题党。
- 抖音：标题可以更口语，但不能制造焦虑。
- B站：简介可以更完整，适合补来源和时间线。
- 小红书：标题可偏经验分享，但要避免虚假承诺。
