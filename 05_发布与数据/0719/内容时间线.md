# 0719 成品视频内容时间线

## 视频讲了什么

这条视频是在演示如何给 Codex 安装和更换皮肤。主线不是单纯介绍某个插件，而是把“闲鱼上有人卖 99 元的 Codex 皮肤服务”拆开，说明它本质上来自 GitHub 开源项目，然后从安装、查看效果、换成自己的图片、再追问如何做整体 UI 风格统一，完整走了一遍实操过程。

## 简单时间线

| 时间段 | 板块 | 内容提炼 |
|---|---|---|
| 00:00-00:18 | 开头引出 | 最近 Codex 装皮肤很火，闲鱼能卖 99 元，但本质是 GitHub 开源项目。 |
| 00:18-00:45 | 安装演示 | 打开 GitHub 项目，把地址交给 Codex 安装，关闭后等待自动安装完成。 |

## 顶部时间轨迹文案

1. 99元皮肤其实开源
2. Codex 的安装

## 检查

- 第一性原理：观众最需要快速判断“这条视频会讲几个步骤，以及自己能不能照做”，所以顶部轨迹优先做成简洁步骤提示，而不是复杂动画。
- 对抗式审查：如果顶部文字太多，会挡住屏幕录制内容；因此每段只保留 8-11 个字左右，放在黑色顶栏内。
