# 封面与标题工作流

更新日期：2026-07-09

目标：把封面和标题前置到发布资料包之前，而不是等视频做完临时起名。

## 文件命名

```text
YYYY-MM-DD_第N期_封面标题方案.md
```

可直接复制模板：

```text
TEMPLATE_封面标题方案.md
```

## 标题方向

每期至少给 4 类标题：

1. 信息型：直接告诉用户发生了什么。
2. 利益型：告诉用户这和自己有什么关系。
3. 判断型：突出“小孙筛选后的判断”。
4. 轻悬念型：有好奇心，但不夸大。

## 封面组成

优先组合：

- 真人表情 / 口播截图。
- 当期最重要新闻的强关联视频画面或官方页面。
- 6-12 字主标题。
- 1 个关键词角标，例如 `AI语音`、`智能体`、`国产模型`。

## 封面原则

- 不堆 3 条新闻，封面只打 1 个主钩子。
- 不写“爆了”“炸裂”“颠覆”这类空词，除非事实真的支撑。
- 封面文字必须和视频第一条或最重要新闻对应。
- 如果使用官方视频截图或产品页面，素材来源要在当期素材台账里记录。
