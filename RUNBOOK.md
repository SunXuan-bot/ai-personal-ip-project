# 运行手册

## 新对话启动

每次继续本项目时，先读：

1. `AGENTS.md`
2. `PROJECT_STATE.md`
3. `DECISIONS.md`
4. `DISCUSSION_QUEUE.md`
5. `REFERENCES.md`
6. `RUNBOOK.md`

然后用 5-10 行说明：

- 当前目标是什么。
- 最近完成了什么。
- 现在最合理的下一步是什么。
- 哪些事情需要用户决策。

## 接收新资料

1. 判断资料类型：人物背景、竞品案例、平台数据、脚本素材、商业资料、灵感碎片。
2. 敏感资料放入 `02_素材与资料/私密参考/`。
3. 未整理资料放入 `02_素材与资料/待整理/`。
4. 可公开引用资料放入 `02_素材与资料/` 下对应主题文件。
5. 在 `REFERENCES.md` 补充资料入口。
6. 如果资料改变了项目判断，更新 `PROJECT_STATE.md` 或 `DECISIONS.md`。

## 做账号定位

每个定位方案必须包含：

- 一句话定位。
- 目标受众。
- 用户为什么会关注。
- 账号主的可信来源。
- 可持续栏目。
- 第一性原理拆解。
- 对抗式审查。
- 最小内容实验。

## 做选题

每个选题至少检查：

- 它解决什么具体问题。
- 它面向谁。
- 它和账号定位有什么关系。
- 它能不能在 60-120 秒讲清楚。
- 它有没有可验证的信息来源。
- 它是否涉及不能公开的隐私或商业信息。

## 做复盘

发布后记录到 `05_发布与数据/`，至少包含：

- 发布时间。
- 平台。
- 标题。
- 选题类型。
- 播放、完播、点赞、评论、转发、关注。
- 主观复盘。
- 下一条怎么改。

## 做资讯视频包装

1. 默认先按横屏 16:9 设计主视觉，画布为 1920×1080。
2. 需要片头、章节转场、观点卡、标题帧、新闻卡片、PPT 重设计时，可使用 `frontend-slides` 技能生成单 HTML 包装页或视觉小样。
3. `frontend-slides` 产出的画面优先作为视觉系统、静态帧、包装页或素材输入，不直接承担完整视频时间轴。
4. 成片编排、口播同步、字幕、B-roll、音频和 MP4 导出继续交给 HyperFrames、Remotion、KrillinAI、剪辑软件或 `ffmpeg`。
5. 适配抖音、视频号等竖屏平台时，优先保留横屏主画面，在 9:16 画布中加入上下视频背板、标题区、字幕区或信息补充区。
6. 每次确定新包装风格后，把可复用的字体、配色、版式、示例帧和适用栏目记录到项目资料中。

## 做 AI 资讯日更

优先读：

1. `00_项目管理/AI资讯日更端到端工作流与分工.md`
2. `04_脚本与内容/AI资讯日更/每日AI资讯日更工作流.md`
3. `04_脚本与内容/AI资讯日更/AI资讯日更_资讯选择与文案优化记录.md`
4. `02_素材与资料/资讯素材/README_资讯视频素材工作流.md`
5. `05_发布与数据/发布资料包/README_发布资料包规范.md`
6. `05_发布与数据/封面与标题/README_封面与标题工作流.md`

默认分工：

- Codex：候选池、来源核验、口播稿、强关联视频素材下载、素材台账、封面标题方案、发布资料包、复盘文档。
- 用户：选题最终拍板、真人录制、剪辑软件中的最终剪辑、字幕人工校对、发布平台和发布时间确认、最终发布。
- 需要先问用户：下载非官方视频、使用争议素材、自动发布、移动或删除大量本地文件。

每期最低交付：

```text
候选池与口播草稿
资讯素材包
封面标题方案
发布资料包
发布复盘
```

## AI 新闻自动筛选定时任务

本机已设置 macOS LaunchAgent：

```text
com.sunxuan.ai-news-scan
```

运行时间：

```text
每天 10:00
每天 17:00
```

输出目录：

```text
04_脚本与内容/AI资讯日更/自动筛选/
```

查看说明：

```text
00_项目管理/AI新闻自动筛选定时任务.md
```

使用方式：

1. 用户看自动生成的 `YYYY-MM-DD_1000/1700_AI新闻候选筛选.md`。
2. 用户反馈哪些像、哪些不像、哪些不够大、哪些不好讲。
3. Codex 把反馈写回 `AI资讯日更_资讯选择与文案优化记录.md`。
4. 如确认主新闻，再进入视频素材包、口播、封面标题和发布资料包流程。

## 外部动作边界

以下动作必须先问用户：

- 公开发布内容。
- 对外发送消息、邮件、私信。
- 任何付费、采购、转账、投资相关动作。
- 删除、覆盖、移动大量文件。
- 暴露个人隐私、家庭、感情、财务、健康信息。
