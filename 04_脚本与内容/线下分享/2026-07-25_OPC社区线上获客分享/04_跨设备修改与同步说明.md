## 在另一台电脑上开始修改

1. 拉取仓库最新内容：

```bash
git pull origin main
```

2. 确认另一台电脑已安装 Dashi PPT 技能。若未安装，可让 Codex 安装 `chuspeeism/dashi-ppt-skill`。

3. 用 Dashi 预览服务打开 `ppt/` 目录。Windows PowerShell 示例：

```powershell
node "$HOME\.codex\skills\dashi-ppt\project\scripts\serve-preview-https.mjs" ".\04_脚本与内容\线下分享\2026-07-25_OPC社区线上获客分享\output\opc-business-sharing-theme01-v3-green-editable\ppt" 5527
```

4. 在系统浏览器打开：

```text
http://127.0.0.1:5527/
```

直接双击 `index.html` 可以查看，但浏览器内编辑不会自动保存；需要保存修改时应使用 Dashi 预览服务。

## 修改边界

- 普通标题、正文、漏斗文字优先在 `goal.json` 或 Dashi 页面编辑器中修改。
- 第 3–6 页和第 8 页使用局部覆盖，相关布局在 `custom/green-editable.css` 和 `custom/green-editable.js` 中。
- 如果重新运行 Dashi 生成器覆盖了 `ppt/index.html`，必须再次执行：

```bash
node output/opc-business-sharing-theme01-v3-green-editable/postprocess-green-editable.mjs
```

- `postprocess-green-editable.mjs` 需在 `2026-07-25_OPC社区线上获客分享/` 目录中执行。
- 不要删除 `ppt/assets/`。最终 HTML 的字体、编辑器、导出功能、图片和二维码都依赖其中的文件。
