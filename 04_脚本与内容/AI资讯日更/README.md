# AI 资讯日更文件夹说明

本文件夹存放“快速筛一遍今天 AI 重要变化”的脚本、候选池、口播稿和文案优化记录。当前默认 5-6 条，质量不足时少讲。

## 当前文件

- `每日AI资讯日更工作流.md`：稳定工作流和口播规则。
- `AI资讯日更_资讯选择与文案优化记录.md`：用户后续反馈、好样本、选题与正文文案优化。
- `自动筛选/`：每天 10:00、17:00 自动生成的 AI 新闻候选筛选稿，用于训练筛选标准。
- `YYYY-MM-DD_第N期_候选池与口播草稿.md`：每期候选池、筛选理由、核验状态、口播草稿；默认 5-6 条，质量不足时少讲。
- `YYYY-MM-DD_第N期录制文本与字幕校对稿.md`：录制后用于字幕和剪辑对齐的文本。

## 命名规则

```text
YYYY-MM-DD_第N期_候选池与口播草稿.md
YYYY-MM-DD_第N期_录制文本与字幕校对稿.md
```

## 和其他目录的关系

- 视频素材放到：`02_素材与资料/资讯素材/YYYY-MM-DD_第N期/`
- 封面标题放到：`05_发布与数据/封面与标题/`
- 发布资料包放到：`05_发布与数据/发布资料包/`
- 发布后数据放到：`05_发布与数据/发布复盘/`
