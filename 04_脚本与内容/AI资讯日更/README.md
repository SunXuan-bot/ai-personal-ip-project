## 使用方式

1. 从候选线索中选择真正与内容人、运营人员或小团队有关的一条。
2. 回到官方源、原视频、GitHub、论文或原作者页面核验。
3. 不做新闻播报式搬运，必须加入大众化解释和自己的判断。
4. 进入正式制作后，在 `04_脚本与内容/` 为该条内容单独建稿，不继续堆进本目录。

## 判断标准

- 主体、动作、变化和影响对象清楚。
- 10-20 秒能解释为什么值得知道。
- 有原始来源和可用画面。
- 能转化为普通内容人或小团队的具体启发。
- 不为了凑数量加入低价值新闻。
