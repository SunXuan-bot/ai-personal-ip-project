# 节目样片制作工作表

日期：2026-07-06

## 1. 样片目标

先做一期可看的测试样片，用来验证：

- 账号定位是否能讲得自然。
- 用户本人出镜和口播状态是否适配。
- 观众能不能理解账号为什么从旅游 / 内容转向 AI。
- 后续是否适合进入 7 天冷启动测试。

## 2. 当前样片方向调整

用户判断：刚开始不会有特别多流量，不需要在前面做太多账号解释，应当直接做内容，让用户先看到账号价值。

因此第一条内容优先从以下方向里选：

- AI 资讯：筛一条或三条普通人值得关注的 AI 信息。
- 大佬访谈：拆一个有启发的 AI 观点。
- 实践心得：讲一个自己真实使用 AI 的小经验。

账号亮相内容可作为备选，或压缩进视频开头 / 结尾一句话。

## 3. 原账号亮相选题备选

> 我为什么一个做旅游内容的人，开始认真学 AI？

这个选题适合作为第一期样片，因为：

- 它能解释账号转向。
- 它不依赖大量外部资料。
- 它能承接旧账号粉丝疑问。
- 它能自然引出“普通人学 AI”和“武汉本地 AI 连接”。

## 4. 样片规格建议

| 项目 | 建议 |
| --- | --- |
| 时长 | 45-75 秒 |
| 形式 | 真人出镜口播优先 |
| 平台 | 抖音 / 视频号可同稿微调 |
| 制作复杂度 | 轻制作，先不要重剪辑 |
| 画面 | 人像口播 + 少量字幕重点 |
| 结尾 | 轻引导关注，不急着强卖社群 |

## 5. 账号亮相内容结构备选

```text
开头 3 秒：
我原来主要做旅游内容，最近开始认真学 AI。

转折：
不是因为我突然成了 AI 专家，而是我发现，AI 可能会改变内容人、小团队和本地企业的很多工作方式。

身份：
我现在一边接触 AI 公司里的真实工作，一边也像普通人一样从 0 到 1 学。

承诺：
接下来我会用人话筛 AI 信息，记录普通人怎么学 AI，也关注武汉本地 AI 机会。

结尾：
如果你也在学 AI，或者在武汉关注 AI，可以先关注我，我们一起看这件事到底怎么落地。
```
