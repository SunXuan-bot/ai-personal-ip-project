# 资讯与海外内容素材规则

更新日期：2026-07-17
状态：作为海外 AI 转述和选题输入规则，不再作为固定日更视频生产线。

## 目标

为一条内容找到能证明原观点、帮助普通观众看懂的真实素材，同时保留来源和公开边界。

## 来源优先级

1. 官方产品 Demo、发布视频或原作者视频。
2. 官方网页、GitHub、论文、产品文档。
3. 原作者或当事人的一手社交帖子。
4. 可信媒体报道。
5. 自制解释卡，只用于补充判断和边界。

## 最小素材包

```text
source.md          原链接、作者、日期、可信度和公开边界
notes.md           原内容讲了什么、采用什么、为什么采用
clips/             只保留最终采用的短片段
screenshots/       只保留最终采用的关键截图
```

视频、图片和音频默认不进入 GitHub；`source.md` 和 `notes.md` 可以同步。

## 工作步骤

1. 先确认内容要表达的唯一判断。
2. 回到原视频、官方页面或原作者链接核验。
3. 长视频先看字幕和 contact sheet，再选择与判断直接相关的时间码。
4. 页面截图只截与主体相关的区域，保留能识别来源的上下文。
5. 记录采用和放弃原因，不用“有画面”替代“画面相关”。
6. 发布前再次检查标题、口播和画面是否表达了同一件事。

## 公开边界

- 官网嵌入 Demo 只能称“官网 Demo”，不能称本地实测。
- GitHub 页面录屏只能称仓库 / README 证据，不能称产品实际运行。
- 社交帖子只能证明发言或线索，不自动证明产品能力。
- 第三方媒体素材不能冒充官方素材。
- 不绕过付费、登录、DRM 或明确限制下载的内容。

旧 AI 资讯视频包装和素材实验的完整保留结论见：

```text
06_复盘与实验/历史实验结论/2026-07_AI资讯视频包装实验结论.md
```
